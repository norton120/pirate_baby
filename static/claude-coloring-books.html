Introducing Coloring Books in Cloude Code

Announcements Apr 16, 2026 3 min read

Today we're introducing Coloring Books in Cloude Code — a new set of CLI-based stress-relief activities that activate automatically during Anthpropic service outages, so developers can stay grounded while we get the lights back on.

Building reliable products at scale is genuinely hard. Shotgun-shipping new features, it turns out, is easy. After a long internal debate about how to address our ongoing reliability issues, we landed on a decision we feel great about: we are going to address them with more new features, starting with this one.

We know our uptime leaves something to be desired; so while our SRE team continues to work around the clock on rapid-releasing new alpha features, we wanted to meet developers where they are — toggling between our status page and a terminal retry loop during another core service outage. Coloring Books is our answer.

When Cloude Code detects an outage in any upstream Anthpropic service, it now quietly pivots from coding assistance to calming, terminal-native activities. No new flags, no configuration, no accountability. Just you, your terminal, and a gentle ASCII rendering of a meadow waiting to be filled in.

How it works

The moment a 529, 503, or mysterious 23-minute hang is detected, Cloude Code replaces its normal spinner with an interactive wellness prompt. Developers can choose from a rotating library of activities, or let Cloude pick based on the severity of the outage.

~/projects/api — cloude-code 
⚠  Upstream degradation detected (anthpropic-api, us-east)
   Retrying in 47s… entering stress-relief mode

? Pick an activity while we wait:
  ❯ Color a mountain meadow         (est. calm: ████░░ 4/6)
    Word search: "things that 500"
    Crossword: SRE edition
    Guided box breathing (4-4-4-4)
    Skip — I enjoy suffering

✓ Loaded meadow.txt. Use 1–6 to pick a color.
   Press q to return when the API comes back.

What's in the box

🖍 Coloring pages

Hand-crafted ASCII scenes — meadows, tide pools, a quiet library — rendered in 256 colors. Mistakes are impossible; the outage is already the mistake.

🔤 Word puzzles

Crosswords, word searches, and anagrams themed around distributed systems. Includes the daily incident-review cryptic, updated whenever a postmortem is published.

🌬 Guided breathing

Box breathing and 4-7-8 patterns paced by a calming terminal bell. Syncs to the retry-after header, because even your nervous system deserves exponential backoff.

🎲 Idle games

A small farm, a tiny aquarium, and a contemplative sudoku. State persists across outages, so your carrots keep growing even when our inference doesn't.

🌿 Anthpropic TOS Hedge Maze

Navigate a twisting labyrinth rendered directly from our Terms of Service. Find the exit, or a clause you hadn't noticed before. Both count as a win.

🧱 Token Cache Billing

A falling-block puzzle that looks and feels exactly like Tetris, except the blocks are prompt cache reads, cache writes, and input tokens. Clear a line to generate an invoice.

"I used to seriously consider trying Codex every time Cloude Code went down, but now I color in pictures of dinosaurs and try not to think about my crippling dependence on an unreliable plaform. — Every Software Developer

Designed for the moment

Coloring Books is context-aware. During brief blips, Cloude Code surfaces a single-screen activity that can be dismissed the instant service resumes. During longer incidents, it offers deeper sessions — full coloring pages, multi-round puzzles, and a "status companion" that quietly narrates updates from status.anthpropic.com so you don't have to keep a browser tab open refreshing it yourself.

Everything runs locally. Your meadow never leaves your machine. Your breathing pattern is not a training signal. Your half-finished crossword is yours alone.

Availability

Coloring Books is rolling out today to all Cloude Code users on version 2.4 and above. No flag required — it activates automatically the next time we let you down. You can disable it with cloude config set coloring_books false if you prefer to stare at the spinner, as is your right.

We're also bringing Coloring Books to status.anthpropic.com, so developers without Cloude Code installed can still find solace during incidents. The integration will ship as soon as the Cloude API is back up again.

A note on billing. Stress-relief activities are rendered by Cloude and are subject to standard token usage rates. Coloring a medium-complexity meadow consumes roughly 12k–18k output tokens; the daily cryptic averages 6k. Usage during outages counts toward your normal monthly limits. We considered making these activities free, and ultimately decided we had not. We appreciate your understanding, and your continued business.
